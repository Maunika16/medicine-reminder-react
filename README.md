# Medicine Reminder App (React + Vite)

A simple web application to help users set reminders for medicines using a clean React interface.

## Features
- Add medicine name
- Set reminder time
- View reminders list
- Simple and responsive UI

## Tech Stack
- React
- Vite
- JavaScript
- CSS

## Project Structure
src/
 components/
   ReminderForm.jsx
   ReminderList.jsx
 styles/
   app.css

## How to Run Locally

1. Install dependencies
npm install

2. Start development server
npm run dev
